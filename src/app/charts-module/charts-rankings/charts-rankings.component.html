O que são os dados de ranking

Os dados são divididos em categorias:

Os mais atrasados

Os mais atrasados, são nuermos que estão por um determinado periodo sem cair. Usando uma forma simples de amostragem, em valor X de corcursos. Pode se dizer que são as bolinhas mais leves;

Os que mais caem

Os que mais caem são numeros que possuem uma presença mais elevada por N concursos, dando a sensação de peso, mesmo que imaginario remete a bolinhas mais pesadas.

Os Reinciadentes

Os numeros reicidentes são numeros que incistem em cair, ou seja diferente dos que mais caem, os reincidentes são mostrados quantas ocorrencia seguida por um intervalo,porem diferente dos que mais caem, embora serão nostrados nos graficos dos que mais caem, neste grafico mostra quantas vezes estão caindo em relação ao concurso atual

query_stats Média

A média é uma medida estatística que representa o valor típico ou central de um conjunto de números. Ela é calculada somando todos os valores e dividindo o resultado pelo número total de elementos no conjunto. A média fornece uma representação equilibrada do conjunto de dados, tornando-a uma medida útil para resumir e comparar conjuntos numéricos.



Linha de tendencia

Uma linha de tendência é uma representação gráfica que visa mostrar a direção geral em que os dados de um conjunto estão se movendo. Ela é frequentemente utilizada em gráficos para destacar padrões, tendências ou relações em dados ao longo do tempo. Uma linha de tendência pode ser linear, exponencial, polinomial, entre outros tipos, dependendo da natureza dos dados e do comportamento esperado.
