query_stats Média


A média é uma medida estatística que representa o valor típico ou central de um conjunto de números. Ela é calculada somando todos os valores e dividindo o resultado pelo número total de elementos no conjunto. A média fornece uma representação equilibrada do conjunto de dados, tornando-a uma medida útil para resumir e comparar conjuntos numéricos.




Linha de tendencia


Uma linha de tendência é uma representação gráfica que visa mostrar a direção geral em que os dados de um conjunto estão se movendo. Ela é frequentemente utilizada em gráficos para destacar padrões, tendências ou relações em dados ao longo do tempo. Uma linha de tendência pode ser linear, exponencial, polinomial, entre outros tipos, dependendo da natureza dos dados e do comportamento esperado.
